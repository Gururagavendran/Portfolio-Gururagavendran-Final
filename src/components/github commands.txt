Setting up a repository:
git init
git clone <repository-url>

When updating local repo changes to github:
git pull origin main
git add .
git commit -m "Updated feature / fixed bug"
git push origin main

Checking git status:
git status

Adding, checking and deleting remote access:
git remote add origin <GitHub URL>
git remote -v
git remote remove origin (Shortened: git remote rm origin)

if any conflict in pushing (branch diverged):
git pull origin main --rebase
git push origin main
