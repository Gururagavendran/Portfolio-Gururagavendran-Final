🚀 Getting Started

To run this project locally:

# Clone the repository
git clone https://github.com/gururagavendran/Portfolio-Gururagavendran-Final.git

# Navigate to project folder
cd Portfolio-Gururagavendran-Final

# Install dependencies
npm install

# Start development server
npm run dev
